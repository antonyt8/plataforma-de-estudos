Substantivo

SUBSTANTIVO

Substantivo e o nome que se da aos seres, sejam reais ou imaginarios, animados ou inanimados.

Atencao: Qualquer palavra precedida de artigo ou de outro determinante transforma-se em substantivo.

Ex.: Pediu para a menina cantar. (verbo) / O cantar da menina e belo. (substantivo)

Classificacao dos substantivos

  • Proprios: denominam um unico ser de um conjunto. Ex.: Alexandre, Paulo, Maria, Joana.
  • Comuns: denominam os seres da mesma especie. Ex.: cidade, cadeira, pais, praia.
  • Concretos: indicam seres de existencia independente. Ex.: lua, pedra, Deus, fada, duende.
  • Abstratos: indicam seres de existencia dependente (acoes, estados, qualidades, sentimentos). Ex.: vida, beleza, frio, amor.
  • Simples: formados por um radical. Ex.: chuva, couve.
  • Compostos: formados por mais de um radical. Ex.: guarda-chuva, couve-flor.
  • Primitivos: nao se originam de outra palavra. Ex.: pedra, mar.
  • Derivados: originam-se de primitivo. Ex.: pedreira, maresia.
  • Coletivos: no singular, indicam conjunto. Ex.: cardume.

Genero

Substantivos uniformes

  • Sobrecomuns: uma so forma para ambos os sexos. Ex.: a crianca.
  • Comuns de dois generos: variacao do determinante. Ex.: o/a jornalista.
  • Epicenos: designam animais sem variacao de artigo. Ex.: a cobra macho / a cobra femea.

Substantivos biformes

Apresentam uma forma para o masculino e outra para o feminino. Ex.: homem/mulher, gato/gata.

Grau

Os substantivos podem se flexionar no aumentativo ou no diminutivo.

Ex.: mae -> maezinha -> maezona | pai -> paizinho -> paizao

Aumentativos sinteticos (selecionados)

  • casa -> casarao
  • boca -> bocarra
  • nariz -> narigao
  • voz -> vozeirao
  • sapato -> sapatorra/sapatao
  • mestre -> mestraco*
  • ministro -> ministraco*
  • medico -> medicastro*

Diminutivos sinteticos (selecionados)

  • casa -> casita/casebre/casinhola
  • rio -> riacho
  • rua -> ruela/rueta
  • sala -> saleta
  • ovo -> ovulo
  • velho -> velhote*
  • papel -> papelucho*
  • rei -> regulo*

* Formas com possivel sentido pejorativo.

Numero

Regra geral: acrescenta-se -s para formar o plural.

Ex.: casa/casas; mae/maes; pai/pais.

Casos importantes

  • Diminutivos em -zinho: pluraliza-se a base e depois o sufixo. Ex.: bar -> bares -> barezinhos.
  • Plural metafonico: mudanca de timbre da vogal tonica em alguns substantivos.
  • Compostos sem hifen: plural regular. Ex.: aguardentes.

Compostos com hifen

  • So o 1o varia: pe-de-meia -> pes-de-meia; joao-de-barro -> joes-de-barro.
  • So o 2o varia: guarda-roupa -> guarda-roupas; alto-falante -> alto-falantes.
  • Ambos variam: tenente-coronel -> tenentes-coroneis; guarda-civil -> guardas-civis.
  • Invariaveis: perde-ganha, bota-fora, quebra-nozes.
  • Admitem mais de um plural: padre-nosso -> padre-nossos/padres-nossos.
